🔆 big-light

ビッグライトモード

文字そのものがUI。リンクに手触りがある。老人と子供が先に使えて、大人が追いつくUI。

→ いま、この画面の右上に ☀️ / 🔆 のトグルが出ています。押すと このページ自身がビッグライトモードに切り替わります（Alt+B でも）。これがそのまま導入後の動きです。

CDN で導入

本文末尾に 2行 足すだけ。右上にトグルが出現します。

<link rel="stylesheet" href="https://osakenpiro.github.io/big-light/v1/big-light-v0.css">
<script src="https://osakenpiro.github.io/big-light/v1/big-light-v0.js"></script>

仕様

  • [data-biglight="1"] が root に付いたときだけ発火 — 既存デザインへの影響ゼロ
  • 右上固定トグル（☀️ / 🔆）— クリック or Alt+B で切替
  • 状態永続化 — localStorage osakenpiro.biglight.v1
  • クロスオリジン引継ぎ — URLパラメータ ?bl=1
  • JS API — window.BigLight.{on,off,toggle,isOn}

設計思想

  • Boolean → Float の UI実装 — ダーク / ライトの二値に「密度軸」を一本足す
  • 文字そのものがUI — カード・パネル等の装飾コンテナを排し、テキスト自体をタップ対象に
  • 腐葉土設計 — 既存デザインを壊さず、上に重ねる規格化レイヤー
